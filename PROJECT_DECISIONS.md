# Project Decisions & Design System Specification

## Brand Positioning
- **Target Audience:** Technical recruiters, Engineering Managers, and Open Source collaborators in the AI/ML and Web space.
- **Brand Personality:** Minimal, Professional, Future-Focused, Confidence without Hype.

## Design System Tokens
- **Primary Accent:** `#3B82F6` (Electric Blue)
- **Canvas Background:** `#0B0B0F` (Rich Deep Black/Blue)
- **Card Background:** `#0A0A0A`
- **Border Subtle:** `#1F2028` (Subtle dark separator)
- **Text Primary:** `#F3F4F6` (Off-white display)
- **Text Secondary:** `#9CA3AF` (Muted Zinc)

## Grid & Spacing
- **Base Grid:** 4px (Strict 4pt system)
- **Maximum Width:** 800px (Optimized for GitHub center column)
- **Gutters:** 16px (gutter spacing between blocks)

## Typography Hierarchy
- **Display Name:** 28px Bold, letter-spacing -0.5px
- **Professional Title:** 15px Semibold, letter-spacing -0.1px
- **Body & Statements:** 14px Regular
- **Section Headers:** 12px Semibold Uppercase, letter-spacing 1px

## Component Architecture (V3 Narrative)
1. `hero.svg` (Name, Role, Brand Statement)
2. `currently-building.svg` (SkillTrove active tracking)
3. `journey.svg` (Machine Learning to Agentic AI path)
4. `tech-stack.svg` (AI Engineering, Software Engineering, Developer Toolkit)
5. `projects.svg` (Single-column layout for Forensic Analyzer, ClimateSCOMC)
6. `philosophy.svg` (Vision statement and architectural direction)
7. `principles.svg` (Core principles: Solve Real Problems, Simplicity Scales, User-Centric, Ship and Iterate)
8. `footer.svg` (Portfolio, LinkedIn, Email anchors)
